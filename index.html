Configurar Supabase 🔐

1. Ve a https://supabase.com

2. Crea un proyecto nuevo

3. Ya configurado ✔️ (esta app ya usa tus claves)

4. Ejecuta este SQL en Supabase

create table profiles ( id uuid primary key references auth.users(id), name text, avatar_url text, role text default 'user' );

5. Trigger automático

create function handle_new_user() returns trigger as $$ begin insert into profiles (id) values (new.id); return new; end; $$ language plpgsql; create trigger on_auth_user_created after insert on auth.users for each row execute procedure handle_new_user();

6. Activar seguridad

alter table profiles enable row level security;
