Guía Supabase ⚙️

OAuth Server

Tu app es ahora un Authorization Server OAuth 2.1. Terceros pueden usar "Sign in with Nuestro Hogar Común".

Google OAuth

Dashboard → Auth → Providers → Google. Client ID + Secret desde Google Cloud Console.

Apple OAuth

Dashboard → Auth → Providers → Apple. Service ID + Private Key desde Apple Developer.

OTP Email

Funciona sin configuración extra. Supabase envía el código automáticamente.

OTP SMS

Configura Twilio/MessageBird en Dashboard → Auth → Providers → Phone
